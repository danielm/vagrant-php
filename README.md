# Simple Vagrant box for PHP Development

Simple Vagrant project for LAMP development.

## Includes:
- Ubuntu 16.04 LTS (Xenial 64)
- Apache 2
- MySQL
- PHP 7
- phpMyAdmin
- Git
- Composer

## Details
 - Default MySQL/phpMyAdmin info:
   - user: root
   - password: 1234567
 - Default document root:
   - src/public/
